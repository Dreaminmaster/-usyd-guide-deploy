# USYD 新生手册

面向悉尼大学语言班（CET/DEC）及正式学位国际新生的中文非官方操作指南。

网站重点覆盖：

- 入学时间线与可保存清单
- CET/DEC 语言班
- Canvas、MyUni、Sydney Student
- 数字学生证与 Apple Wallet
- UniKey、Okta、校园 Wi‑Fi
- 抵达悉尼后的住宿、交通、银行与安全事项

> 所有个人 Offer、学校邮件和悉尼大学官方系统中的信息，优先级均高于本网站。

## 维护

网站为纯静态页面。`site.html.gz.b64` 保存压缩后的完整页面，`index.html` 负责在浏览器中加载并解压；GitHub Actions 自动发布到 Pages。
